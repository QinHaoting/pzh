## 项目准备

1. 首先加载`./pom.xml`，加载依赖
2. 然后创建一个名为`pzh_db`数据库，然后导入`.\src\main\java\com\pzh\sql\pzh_db.sql`数据库脚本文件
3. 再去`.\src\main\resources\application.yml`改其中的数据库配置，如用户名、密码
4. 最后**IDEA**项目运行即可
